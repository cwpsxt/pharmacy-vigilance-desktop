# AI 智能助手功能实现文档

## 一、功能概述

在现有 AI 助手基础上，扩展以下智能分析功能：

| 功能模块 | 描述 | 数据安全级别 |
|---------|------|-------------|
| 📊 数据洞察分析 | 分析统计数据，发现规律和异常 | ✅ 安全（仅统计值） |
| 📈 趋势预测预警 | 预测不良反应趋势，异常预警 | ✅ 安全（仅统计值） |
| 📝 智能报告生成 | 自动生成分析报告文本 | ✅ 安全（仅统计值） |
| 💊 用药安全建议 | 基于数据给出专业建议 | ✅ 安全（仅药品名+统计） |
| ❓ 通用问答 | 药物警戒知识问答（现有功能） | ✅ 安全（无业务数据） |

## 二、数据安全设计

### 2.1 敏感数据隔离原则

**绝对禁止发送给AI的字段：**
```python
SENSITIVE_FIELDS = [
    'report_code',           # 报告表编码
    'medical_record_no',     # 病历号/门诊号
    'reporter_signature',    # 报告人签名
    'import_batch_id',       # 导入批次ID
]
```

**允许发送给AI的数据（仅聚合统计）：**
```python
SAFE_DATA = {
    # 统计数值
    'total_count': 150,              # 总报告数
    'severe_count': 30,              # 严重报告数
    'general_count': 120,            # 一般报告数
    
    # 药品统计（不含个人信息）
    'drug_statistics': [
        {'drug_name': '阿莫西林', 'count': 25, 'severe_rate': 0.12},
        {'drug_name': '头孢类', 'count': 18, 'severe_rate': 0.22},
    ],
    
    # 时间趋势（仅数量）
    'monthly_trend': [
        {'month': '2024-01', 'count': 45},
        {'month': '2024-02', 'count': 52},
    ],
    
    # 分类统计
    'category_distribution': {
        '抗生素类': 45,
        '心血管类': 30,
    },
    
    # 不良反应类型统计
    'reaction_statistics': [
        {'reaction_name': '皮疹', 'count': 35},
        {'reaction_name': '恶心呕吐', 'count': 28},
    ]
}
```

#### 功能1：数据洞察分析

**触发方式：** 点击「数据洞察」按钮 或 输入 `/分析`

**处理流程：**
```
1. 前端调用 /api/ai/analyze 接口
2. 后端查询数据库，计算统计指标
3. 构建安全数据包（不含敏感信息）
4. 调用 DeepSeek API 分析
5. 返回洞察结果
```

**发送给AI的数据示例：**
```json
{
  "task": "数据洞察分析",
  "statistics": {
    "period": "2024-01 至 2024-06",
    "total_reports": 356,
    "type_distribution": {
      "一般": 245,
      "严重": 56,
      "新的+一般": 38,
      "新的+严重": 17
    },
    "top_drugs": [
      {"name": "阿莫西林胶囊", "count": 28, "severe_rate": "14.3%"},
      {"name": "头孢克肟片", "count": 22, "severe_rate": "18.2%"}
    ],
    "top_reactions": [
      {"name": "皮疹", "count": 45},
      {"name": "恶心", "count": 38}
    ],
    "monthly_trend": [
      {"month": "2024-01", "count": 52},
      {"month": "2024-02", "count": 48}
    ]
  }
}
```

**AI返回示例：**
```markdown
## 📊 数据洞察分析报告

### 关键发现
1. **报告总量**：共356例，严重反应占比20.5%，需重点关注
2. **高风险药品**：头孢克肟片严重反应率18.2%，高于平均水平
3. **常见反应**：皮疹(45例)和恶心(38例)为主要不良反应类型

### 趋势分析
- 2月较1月下降7.7%，整体呈平稳态势

### 建议措施
1. 加强头孢类药品用药监测
2. 完善皮疹类反应的预防指导
```

## 六、安全检查清单

### 发送前检查

- [ ] 数据包不含 `report_code`
- [ ] 数据包不含 `medical_record_no`
- [ ] 数据包不含 `reporter_signature`
- [ ] 数据包不含任何可识别个人身份的信息
- [ ] 仅包含聚合统计数值
- [ ] 药品名称为通用名（非特定批次）

### 代码审计点

```python
def validate_safe_data(data):
    """验证数据安全性"""
    FORBIDDEN_KEYS = [
        'report_code', 'medical_record_no', 'reporter_signature',
        'patient', 'id_card', 'phone', 'address'
    ]
    
    data_str = json.dumps(data, ensure_ascii=False)
    for key in FORBIDDEN_KEYS:
        if key in data_str:
            raise SecurityError(f"数据包含敏感字段: {key}")
    
    return True
```

## 八、注意事项

1. **隐私优先**：任何情况下都不发送敏感数据
2. **本地聚合**：所有统计计算在本地完成
3. **日志脱敏**：日志中不记录敏感信息
4. **用户告知**：明确告知用户AI不访问隐私数据
5. **可追溯**：保留AI分析请求日志（脱敏后）
