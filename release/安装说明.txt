医院药学药物警戒与运营绩效可视化分析系统
====================================================
安装说明 v1.0.0
====================================================

一、系统要求
----------------------------------------------------
- 操作系统：Windows 10 / Windows 11（64 位）
- 内存：建议 4 GB 以上
- 硬盘空间：至少 1 GB 可用空间
- 网络：AI 助手功能需要联网（可选）


二、安装步骤
----------------------------------------------------

1. 双击运行 "PharmacyVigilanceSystem Setup 1.0.0.exe"

2. 如果系统弹出"Windows 已保护你的电脑"窗口：
   - 点击"更多信息"
   - 点击"仍要运行"
   （这是因为安装包未购买代码签名证书，属于正常现象）

3. 如果杀毒软件报警：
   - 选择"信任此程序"或"添加到白名单"
   - 这是 PyInstaller 打包程序常见的误报，文件本身安全

4. 选择安装路径（建议安装到非系统盘，如 D:\Programs\）

5. 勾选"创建桌面快捷方式"，点击"安装"

6. 安装完成后，桌面会出现"药物警戒分析系统"图标


三、首次启动
----------------------------------------------------

1. 双击桌面快捷方式启动程序

2. 首次启动需要 10-15 秒（系统正在初始化后端服务）

3. 出现登录界面后，使用以下账号登录：

       用户名：admin
       密  码：admin

4. 登录成功后请立即修改密码（用户管理 - 修改密码）


四、常见问题
----------------------------------------------------

问：启动后白屏或转圈很久不进入登录页？
答：首次启动需解压后端服务文件，请耐心等待 15-20 秒。
    如果超过 30 秒仍无响应：
    1. 关闭程序，重新启动
    2. 打开任务管理器，检查是否有 server.exe 进程
    3. 关闭其他占用 5001 端口的程序

问：弹出"网络错误，请检查服务是否启动"？
答：可能是 Python 后端未启动成功。请：
    1. 完全关闭程序
    2. 任务管理器结束所有 server.exe 进程
    3. 重新启动程序

问：AI 助手提示无法使用？
答：AI 助手需要联网才能工作。如果在内网环境：
    1. 检查是否能访问 https://api.deepseek.com
    2. 联系管理员开放 API 访问

问：忘记密码怎么办？
答：请联系系统管理员重置数据库，或重新安装程序（会清除所有数据）


五、卸载方法
----------------------------------------------------

方式 1：控制面板 - 程序和功能 - 找到"药物警戒分析系统" - 卸载

方式 2：进入安装目录，运行 Uninstall.exe

注意：卸载会保留用户数据（数据库、上传文件）。
      如需完全清除，请手动删除安装目录。
