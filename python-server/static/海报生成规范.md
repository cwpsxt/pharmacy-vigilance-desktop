4. 导出功能的“避坑”指南 (重要)我们在开发过程中解决了几个严重影响图片导出的 BUG，生成新海报时请务必保留以下逻辑：4.1 文字渐变问题问题：CSS 属性 background-clip: text 在 html2canvas 中兼容性极差，导出后文字会变成矩形色块或消失。解决方案：不要使用背景裁剪做渐变字。使用 color 配合 text-shadow 来模拟发光效果，或者使用 SVG 图片代替文字。4.2 导出错位与偏移问题：当页面有滚动条或动态浮动动画时，点击导出按钮，生成的图片内容会向下偏移或错位。解决方案：使用**“沙箱克隆法”**（Sandbox Clone Strategy）。Clone: 复制要截图的 DOM 节点。Position: 将克隆体放入一个 position: fixed; top: 0; left: 0 的容器中。Freeze: 在截图前移除克隆体内部的 CSS 动画类（如 .animate-float）。Capture: 对克隆体进行截图，而不是原屏幕元素。4.3 图标对齐问题问题：FontAwesome 图标在导出时可能会垂直偏上或偏下。解决方案：给图标的父容器显式增加 Flex 居中样式：.icon-fix {
    display: flex;
    align-items: center;
    justify-content: center;
}
